Arrays

Declaración y uso de Arrays

Los Arrays son una estructura de datos que almacenan una secuencia de valores ordenados. En JavaScript, los arrays pueden contener valores de diferentes tipos, como números, cadenas de texto, objetos, entre otros. 

// Los Arrays son un tipo de dato primitivo que almacena una colección de valores de un mismo tipo.

        const numeros = [10, 20, 30, 40, 50];
                  
        const meses = new Array(
          "Enero",
          "Febrero",
          "Marzo",
          "Abril",
          "Mayo",
          "Junio",
          "Julio",
          "Agosto",
          "Septiembre",
          "Octubre",
          "Noviembre",
          "Diciembre"
        );
                  
        console.log(numeros); // [10,20,30,40,50]
        console.log(meses); /* ["Enero","Febrero","Marzo","Abril","Mayo","Junio","Julio","Agosto",
        "Septiembre","Octubre","Noviembre","Diciembre"] */
                  
//Recordar!!!  los Arrays siempre inician en la posición 0 y terminan en la posición (longitud-1)
                  
//No es necesario que los arreglos esten relacionados, pueden almacener multiples cantidades de informacion no relacionada entre si
                  
        const deTodo = [
        "Hola",
        10,
        true,
        { nombre: "Sebastian", trabajo: "Frontend Developer" },
        [4872344],
        ];

        console.log(deTodo);

Acceder a los valores de un array

Para acceder a los valores de un array en JavaScript, se utiliza su índice, comenzando desde 0 para el primer elemento. 

//Acceder a los valores de un array

        const numeros = [10, 20, 30, 40, 50, [1, 2, 3, 4]];

//Se esta manera se accede a la posicion del array

        console.log(numeros[0]); // 10
        console.log(numeros[1]); // 20
        console.log(numeros[2]); // 30
        console.log(numeros[3]); // 40
        console.log(numeros[4]); // 50
        console.log(numeros[5][0]); // 1
                  
//Acceder a un elemento del array que no existe devuelve undefined

Acceder a todos los elementos de un array

Para acceder a todos los elementos de un array en JavaScript, se puede utilizar bucles como for ,for...of , o métodos como .forEach(). Veamos un ejemplo utilizando el bucle for: 

//Como acceder a todos los elementos de un array

          "Enero",
          "Febrero",

          "Agosto",

        console.log(meses.length); //12
                  
        for (let i = 0; i < meses.length; i++) {
          console.log(`El mes de ${meses[i]}`);

Añadir nuevos valores a un array

Para añadir nuevos valores a un array en JavaScript, se puede usar el método .push() para agregarlos al final o .unshift() para agregarlos al inicio. 

//Añadir nuevos elementos a un array

        const carrito = [];
                  
//Definir un producto
                  
        const producto = {
          nombre: "Monitor 20 pulgadas",
          precio: 400,
        };
                  
        const movil = {
          nombre: "iPhone 12 Pro",
          precio: 1000,
        };
                  
        const teclado = {
          nombre: "Teclado Macbook Pro",
          precio: 150,
        };
                  
        carrito.push(producto); //Añadir al final del array
        carrito.push(movil); //Añadir al final del array
        carrito.unshift(teclado); //Añadir al principio del array
                  
        console.table(carrito);
                  
        ┌─────────┬───────────────────────┬────────┐
        │ (index) │ nombre                │ precio │
        ├─────────┼───────────────────────┼────────┤
        │ 0       │ 'Teclado Macbook Pro' │ 150    │
        │ 1       │ 'Monitor 20 pulgadas' │ 400    │
        │ 2       │ 'iPhone 12 Pro'       │ 1000   │
        └─────────┴───────────────────────┴────────┘

Añadir elementos con spread operator

En JavaScript, para añadir nuevos elementos a un array sin modificar el original, puedes usar el spread operator (...). Esto permite crear una nueva copia del array existente y añadirle los nuevos elementos, manteniendo la inmutabilidad. Es especialmente útil cuando trabajas con estados en aplicaciones modernas. 

// Crear un array vacío

        const carrito = [];
                  
// Definir productos como objetos

        const producto = { nombre: "Monitor 20 pulgadas", precio: 400 };
        const movil = { nombre: "iPhone 12 Pro", precio: 1000 };
        const teclado = { nombre: "Teclado Macbook Pro", precio: 150 };
                  
// Añadir productos al array utilizando el spread operator

        let resultado;
                  
// Añadir el primer producto

        resultado = [...carrito, producto]; 
                  
// Añadir otro producto al final

        resultado = [...resultado, movil];
                  
// Añadir un producto al inicio

        resultado = [teclado, ...resultado];
                  
// Mostrar el resultado final

        console.table(resultado);

        ┌─────────┬───────────────────────┬────────┐
        │ (index) │ nombre                │ precio │
        ├─────────┼───────────────────────┼────────┤
        │ 0       │ 'Teclado Macbook Pro' │ 150    │
        │ 1       │ 'Monitor 20 pulgadas' │ 400    │
        │ 2       │ 'iPhone 12 Pro'       │ 1000   │
        └─────────┴───────────────────────┴────────┘

Añadir, eliminar o modificar elementos de un array

Los arrays en JavaScript permiten manipular sus elementos mediante métodos integrados como push(), splice(), pop() y shift() Estos métodos son útiles para añadir, eliminar o modificar elementos de forma flexible. 

// Crear un array vacío

        const carrito = [];
                  
// Definir productos como objetos

        const monitor = { nombre: "Monitor 20 pulgadas", precio: 400 };
        const movil = { nombre: "iPhone 12 Pro", precio: 1000 };
        const teclado = { nombre: "Teclado Macbook Pro", precio: 150 };
        const notebook = { nombre: "Acer Nitro 5", precio: 1100 };
        const mouse = { nombre: "Logitech", precio: 80 };
                  
// Agregar productos al array usando el método push (añade al final)

        carrito.push(teclado); 
        carrito.push(monitor);
        carrito.push(movil);
        carrito.push(notebook);
        carrito.push(mouse);
                  
// Mostrar el array con todos los elementos

        console.table(carrito);

        | (index) | nombre               | precio |
        |---------|----------------------|--------|
        | 0       | "Teclado Macbook Pro"| 150    |
        | 1       | "Monitor 20 pulgadas"| 400    |
        | 2       | "iPhone 12 Pro"      | 1000   |
        | 3       | "Acer Nitro 5"       | 1100   |
        | 4       | "Logitech"           | 80     |

                  
// Eliminar un elemento en una posición específica usando splice
// En este caso, eliminamos el primer elemento (índice 0)

        carrito.splice(0, 1); // Elimina 1 elemento desde la posición 0
        console.table(carrito); // Mostrar el array actualizado

      | (index) | nombre               | precio |
      |---------|----------------------|--------|
      | 0       | "Monitor 20 pulgadas"| 400    |
      | 1       | "iPhone 12 Pro"      | 1000   |
      | 2       | "Acer Nitro 5"       | 1100   |
      | 3       | "Logitech"           | 80     |
                  
// Eliminar el último elemento usando pop

        carrito.pop(); // Elimina el último elemento del array
        console.table(carrito); // Mostrar el array sin el último elemento

      | (index) | nombre               | precio |
      |---------|----------------------|--------|
      | 0       | "Monitor 20 pulgadas"| 400    |
      | 1       | "iPhone 12 Pro"      | 1000   |
      | 2       | "Acer Nitro 5"       | 1100   |
                  
// Eliminar el primer elemento usando shift

        carrito.shift(); // Elimina el primer elemento del array
        console.table(carrito); // Mostrar el array sin el primer elemento

      | (index) | nombre               | precio |
      |---------|----------------------|--------|
      | 0       | "iPhone 12 Pro"      | 1000   |
      | 1       | "Acer Nitro 5"       | 1100   |

Destructuring de Arrays

En JavaScript, el destructuring permite extraer valores de objetos o arrays de forma más sencilla y directa. Esto es especialmente útil cuando trabajas con estructuras de datos complejas o necesitas trabajar con valores específicos. 

// Definición de un objeto con propiedades simples y anidadas

        const producto = {
          nombre: "Monitor 20 pulgadas",
          precio: 30,
          disponible: true,
          informacion: {
            peso: "1kg",
            medida: "1m",
          },
        };
                  
// Uso de destructuring para extraer propiedades del objeto
        const { nombre } = producto;
                  
        console.log(nombre); // Salida: "Monitor 20 pulgadas"
                  
// Definición de un array con valores
        const numeros = [10, 20, 30, 40, 50];

// Destructuring simple: Extraer el primer valor y agrupar el resto con spread
const [primero, ...restantes] = numeros;

        console.log(primero); // Salida: 10
        console.log(restantes); // Salida: [20, 30, 40, 50]

Iterar sobre un array de objetos

Cuando trabajamos con arrays de objetos en JavaScript, como un carrito de compras, es común necesitar iterar sobre los elementos para acceder o manipular sus propiedades. Aquí exploraremos distintas formas de realizar esta tarea. 

// Array de objetos

//Este array contiene varios objetos, cada uno representando un producto con las propiedades nombre y precio.

        const carrito = [
        { nombre: "Monitor Samsung 24 pulgadas", precio: 110 },
        { nombre: "Notebook Acer Nitro 5", precio: 1100 },
        { nombre: "Teclado Logitech", precio: 15 },
        { nombre: "Mouse Logitech", precio: 80 },
        { nombre: "Poco X3 Pro", precio: 250 },
        { nombre: "Play Station 4", precio: 150 },
        ];

//Iterar usando un bucle for

        for (let i = 0; i < carrito.length; i++) {
          console.log(`${carrito[i].nombre} - Precio: ${carrito[i].precio}`);
        }

/*El bucle recorre cada índice del array usando i. En cada iteración, accedemos al objeto en carrito[i]
y mostramos sus propiedades nombre y precio con una plantilla de texto.*/

//Iterar usando forEach

        //Funcion tradicional
        carrito.forEach(function (producto) {
          console.log(`${producto.nombre} - Precio: ${producto.precio}`);
        });

        //Arrow function
        carrito.forEach((producto) => {
          console.log(`${producto.nombre}`);
        });
      
//Función tradicional y flecha: Ambas formas recorren el array, ejecutando el código para cada elemento.
//Parámetro del callback: En este caso, el parámetro producto representa cada objeto del array durante la iteración.
//Plantilla de texto: Usamos plantillas de texto (${}) para mostrar las propiedades del objeto de forma clara.

//Con ambos métodos obtenemos el mismo resultado

        Monitor Samsung 24 pulgadas - Precio: 110
        Notebook Acer Nitro 5 - Precio: 1100
        Teclado Logitech - Precio: 15
        Mouse Logitech - Precio: 80
        Poco X3 Pro - Precio: 250
        Play Station 4 - Precio: 150
        
//Ventajas de for: El bucle for te permite un control más detallado sobre el proceso de iteración.
//Ventajas de forEach: Es más conciso y enfocado en recorrer elementos

Metodo .map()

El método .map() es una de las funciones más poderosas en JavaScript para trabajar con arrays. Permite recorrer todos los elementos de un array, aplicar una transformación a cada uno de ellos, y devolver un nuevo array con los resultados de esa transformación. A diferencia de otros métodos como .forEach(), .map() no modifica el array original, sino que crea uno nuevo con los valores procesados. 

//Array de productos en el carrito

        const carrito = [
        { nombre: "Monitor Samsung 24 pulgadas", precio: 110 },
        { nombre: "Notebook Acer Nitro 5", precio: 1100 },
        { nombre: "Teclado Logitech", precio: 15 },
        { nombre: "Mouse Logitech", precio: 80 },
        { nombre: "Poco X3 Pro", precio: 250 },
        { nombre: "Play Station 4", precio: 150 },
        ];
                
/*El método map recorre cada elemento del array y ejecuta una función en cada uno de ellos.
Devuelve un nuevo array con los valores resultantes de esa función.*/

        const resultado = carrito.map(function (producto) {
          return `${producto.nombre} - Precio: ${producto.precio}`;
        });
                
        console.log(resultado);

        [
        "Monitor Samsung 24 pulgadas - Precio: 110",
        "Notebook Acer Nitro 5 - Precio: 1100",
        "Teclado Logitech - Precio: 15",
        "Mouse Logitech - Precio: 80",
        "Poco X3 Pro - Precio: 250",
        "Play Station 4 - Precio: 150"
        ]
              
//Filtrar elementos con map

/*Aunque map no está diseñado específicamente para filtrar elementos, se puede usar junto con condiciones para lograr un efecto similar. 
Sin embargo, filter es un método más adecuado si el objetivo principal es filtrar datos.*/

        const resultado2 = carrito.map(function (producto) {
          if (producto.precio > 100) {
            return `${producto.nombre} - Precio: ${producto.precio}`;
          }
        }).filter(item => item !== undefined); // Eliminar valores undefined

        console.log(resultado2);

        [
          "Notebook Acer Nitro 5 - Precio: 1100",
          "Poco X3 Pro - Precio: 250"
        ]

//Uso de map con valores adicionales (Spread operator)

/*Si necesitas extraer o combinar datos adicionales de los objetos mientras los mapeas, el uso del spread operator (...) es muy útil.*/

        const resultado3 = carrito.map(function (producto) {
          return { ...producto, descuento: producto.precio * 0.1 }; // Añadiendo un descuento al producto
        });

        console.table(resultado3);

        [
          { nombre: "Monitor Samsung 24 pulgadas", precio: 110, descuento: 11 },
          { nombre: "Notebook Acer Nitro 5", precio: 1100, descuento: 110 },
          { nombre: "Teclado Logitech", precio: 15, descuento: 1.5 },
          { nombre: "Mouse Logitech", precio: 80, descuento: 8 },
          { nombre: "Poco X3 Pro", precio: 250, descuento: 25 },
          { nombre: "Play Station 4", precio: 150, descuento: 15 }
        ]

/*map:

Devuelve un nuevo array con los resultados de la transformación de cada elemento.
Ideal para transformar o modificar los elementos de un array.
No afecta el array original.

forEach:

No devuelve nada, solo ejecuta una función en cada elemento del array.
Se usa para realizar acciones sobre los elementos, como mostrar información o modificar el estado de elementos en el array original.*/

/*El método map es útil cuando necesitas transformar un array y obtener un nuevo array 
con los resultados de esa transformación. Es muy versátil y puede ser utilizado tanto 
para formatear los datos como para agregar propiedades nuevas a los objetos dentro de un array.*/
