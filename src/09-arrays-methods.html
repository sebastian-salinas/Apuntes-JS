Arrays Methods

Declaración y uso de Arrays Methods

Los métodos de arrays en JavaScript son funciones predefinidas que permiten realizar diversas operaciones sobre los elementos de un array, como agregar, eliminar, modificar, ordenar o iterar a través de sus valores. 

// Método includes()

/*El método includes() verifica si un valor existe dentro de un array.
En el caso de un array de simples valores (por ejemplo, meses), podemos comprobar si un elemento está presente:*/

        const meses = ["Enero", "Febrero", "Marzo", "Abril"];
        const resultado = meses.includes("Enero"); // Retorna true si "Enero" está en el array
        console.log(resultado);  // true

// Método some() para arrays de objetos

/*El método some() se utiliza para verificar si al menos un elemento dentro de un array cumple con una condición dada. 
Esto es útil especialmente cuando trabajamos con arrays de objetos. 
Por ejemplo, podemos comprobar si existe un producto específico en un carrito de compras:*/

        const carrito = [
          { nombre: "Monitor 27 Pulgadas", precio: 500 },
          { nombre: "Televisión", precio: 100 },
          { nombre: "Tablet", precio: 200 },
          { nombre: "Audífonos", precio: 300 },
        ];

const existe = carrito.some((producto) => producto.nombre === "Teclado");
console.log(existe);  // false, porque no existe un "Teclado" en el carrito

//includes() es más útil para arrays simples, donde estamos buscando un valor directo dentro del array.

/*some() es más adecuado cuando se trabaja con arrays de objetos y queremos verificar si alguno de los 
elementos cumple con una condición específica, como si el nombre de un producto en un carrito es igual a "Teclado".*/

Metodo .findIndex()

El método .findIndex() es una función de los arrays en JavaScript que permite encontrar la posición (índice) de un elemento que cumple con una condición especificada dentro de un array. Este método es especialmente útil cuando necesitas localizar la posición de un valor en un array sin tener que recorrerlo manualmente. 

/*En un array de valores simples, puedes usar .findIndex() para encontrar el índice de un valor específico. 
En el siguiente ejemplo, buscamos el índice de "Abril" en el array meses*/

        const meses = ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio"];

        const posicion = meses.findIndex((mes) => {
          return mes === "Abril";
        });

        console.log(posicion);  // 3

/*El método .findIndex() devuelve el índice del primer elemento que cumple con la condición (en este caso, el mes "Abril").
Si no se encuentra ningún elemento que cumpla la condición, devuelve -1.*/

//Usando .findIndex() en un array de objetos

/*El método .findIndex() también puede usarse en arrays de objetos. Supongamos que tienes un array de productos y
deseas encontrar el índice del producto con el nombre "Tablet":*/

        const carrito = [
          { nombre: "Monitor 27 Pulgadas", precio: 500 },
          { nombre: "Televisión", precio: 100 },
          { nombre: "Tablet", precio: 200 },
          { nombre: "Audífonos", precio: 300 },
          { nombre: "Teclado", precio: 400 },
          { nombre: "Celular", precio: 700 },
        ];

        const valoresCarrito = carrito.findIndex((elemento) => {
          return elemento.nombre === "Tablet";
        });

        console.log(valoresCarrito);  // 2

/*En este caso, findIndex() devuelve el índice del primer objeto que tiene un nombre igual a "Tablet" (el valor 2),
lo que nos indica que "Tablet" se encuentra en la tercera posición del array.*/

//Como funciona .findIndex()

/*Retorno: .findIndex() devuelve el índice del primer elemento que cumple con la condición que le pases.
Si ningún elemento satisface la condición, devuelve -1.*/

/*Útil para arrays de objetos: Es especialmente útil cuando trabajas con arrays de objetos y quieres 
encontrar la posición de un objeto específico basándote en una propiedad.*/

Metodo .reduce()

El método .reduce() en JavaScript es una herramienta muy poderosa que permite reducir un array a un único valor. A diferencia de otros métodos como .map() o .filter(), que generan un nuevo array basado en el original, .reduce() acumula o "reduce" los valores de un array según una operación que definas. 

//Funcionamiento de .reduce()

//El método .reduce() toma dos argumentos:

/* 1 - Una función callback que recibe dos parámetros:
* Acumulador (total): es el valor que se va acumulando a medida que se recorren los elementos del array.
* Elemento actual (producto): es el valor del elemento actual del array sobre el que se está iterando.

/* 2 - Un valor inicial (opcional) que establece el valor inicial para el acumulador. Si no se proporciona, 
el primer valor del array se usará como el valor inicial.*/

//Ejemplo utilizando forEach():

        const carrito = [
          { nombre: "Monitor 27 Pulgadas", precio: 500 },
          { nombre: "Televisión", precio: 100 },
          { nombre: "Tablet", precio: 200 },
          { nombre: "Audífonos", precio: 300 },
          { nombre: "Teclado", precio: 400 },
          { nombre: "Celular", precio: 700 },
        ];

// Calcular el total con forEach (sin reduce)

        let total = 0;

        carrito.forEach((producto) => {
          total += producto.precio;
        });

        const formatear = total.toLocaleString("es-Es", {
          minimumFractionDigits: 2, // Mínimo número de decimales
          maximumFractionDigits: 2, // Máximo número de decimales
        });

        console.log(
          `En tu carrito de compras tienes objetos por un valor de : ${formatear} €`
        );

//Ejemplo utilizando reduce() para simplificar el cálculo:

        let resultado = carrito.reduce((total, producto) => {
          return total + producto.precio;
        }, 0);

        console.log(resultado); // 2200

//Explicación de .reduce():

/*Primer argumento de .reduce(): La función callback recibe dos parámetros: total (el acumulador que guarda el valor acumulado)
y producto (el elemento actual del array).*/

/*En cada iteración, el total va acumulando el valor de producto.precio hasta que recorra todos los elementos del array.*/

/*Segundo argumento de .reduce(): El valor 0 es el valor inicial del acumulador (total). Si no lo 
proporcionamos, .reduce() utilizaría el primer elemento del array como valor inicial.*/

/*reduce(): Es más eficiente para operaciones de acumulación o transformación, ya que devuelve un valor 
único que representa la "reducción" de todos los elementos del array, como la suma en este caso.*/

Método .filter()

El método .filter() en JavaScript es utilizado para crear un nuevo array con todos los elementos que cumplen una condición específica que se define dentro de una función. A diferencia de otros métodos como .map(), que transforman los valores del array, .filter() te permite filtrar los elementos que cumplen con una condición, manteniendo los que coinciden y excluyendo el resto. 

//Sintaxis básica:

        array.filter(function(elemento) {
          return condicion;
        });

/*El método .filter() toma una función de devolución de llamada (callback) que recibe un elemento de la lista 
y devuelve true si debe mantenerse en el nuevo array, o false si debe ser excluido.*/

//Ejemplo detallado

/*Imagina que tienes un carrito de compras con varios productos. Si quisieras filtrar los 
productos de acuerdo a ciertas condiciones, podrías hacerlo de la siguiente manera:*/

        const carrito = [
          { nombre: "Monitor 27 Pulgadas", precio: 500 },
          { nombre: "Televisión", precio: 100 },
          { nombre: "Tablet", precio: 200 },
          { nombre: "Audífonos", precio: 300 },
          { nombre: "Teclado", precio: 400 },
          { nombre: "Celular", precio: 700 },
        ];

        // Filtrar productos con un precio superior a 600

        let resultado = carrito.filter((producto) => producto.precio > 600);

        console.log(resultado); // Resultado: [{ nombre: "Celular", precio: 700 }]

/*En este ejemplo, .filter() crea un nuevo array que solo contiene los productos con un precio 
superior a 600. Si el precio de un producto es mayor que 600, el producto se incluye en el nuevo array.*/

//Mas ejemplos:

        resultado = carrito.filter((producto) => producto.nombre !== "Tablet");

        console.log(resultado);
        /* Resultado: [{ nombre: "Monitor 27 Pulgadas", precio: 500 }, { nombre: "Televisión", precio: 100 }, 
        { nombre: "Audífonos", precio: 300 }, { nombre: "Teclado", precio: 400 }, { nombre: "Celular", precio: 700 }]*/

        resultado = carrito.filter((producto) => producto.nombre === "Tablet");

        console.log(resultado);
        // Resultado: [{ nombre: "Tablet", precio: 200 }]

/*Resumen:

.filter() te permite crear un nuevo array con los elementos que cumplen una condición.

Retorna un array nuevo, dejando intacto el array original.

Filtra los elementos según la condición que se le proporcione y excluye los que no cumplen dicha condición.

Es útil cuando necesitas obtener subconjuntos de un array basados en ciertos criterios, como filtrar productos
por precio, nombre, categoría, etc.*/

//Comparativa con otros métodos:

        .filter() vs .map():

        .map():// Crea un nuevo array con los resultados de aplicar una función a cada elemento.

        .filter(): /*Crea un nuevo array con los elementos que cumplen una condición, no transformando 
        los elementos, sino seleccionando solo aquellos que coinciden con la condición.*/

Metodo .find()

El método .find() en JavaScript se utiliza para buscar el primer elemento de un array que cumpla con una condición específica. A diferencia de métodos como .filter(), que retornan un nuevo array con todos los elementos que cumplen la condición, .find() solo devuelve el primer elemento que cumple con la condición, o undefined si no se encuentra ninguno. 

//Ejemplo detallado con .find():

//Imagina que tienes un carrito de compras con varios productos, y quieres encontrar el primer producto que sea una "Tablet":

        const carrito = [
        { nombre: "Monitor 27 Pulgadas", precio: 500 },
        { nombre: "Televisión", precio: 100 },
        { nombre: "Tablet", precio: 200 },
        { nombre: "Audífonos", precio: 300 },
        { nombre: "Teclado", precio: 400 },
        { nombre: "Celular", precio: 700 },
        ];

// Usando .find para buscar el primer producto que sea una "Tablet"

        const resultado2 = carrito.find((producto) => producto.nombre === "Tablet");

        console.log(resultado2); // Resultado: { nombre: "Tablet", precio: 200 }

//Explicación del código:

        carrito.find(): /* El método .find() recorre el array carrito y evalúa la condición que le pases
        en la función callback. En este caso, busca el primer producto cuyo nombre sea "Tablet".*/

        producto.nombre === "Tablet"/*: Este es el criterio que estamos evaluando. Si el nombre del producto es igual
        a "Tablet", entonces ese producto será el primero que se devuelve por el método .find().*/

/*Resultado: 

El método devuelve el primer producto que cumple con la condición. En este caso, 
devuelve el objeto*/ { nombre: "Tablet", precio: 200 } /*. Si no hubiera ningún producto con 
ese nombre, habría retornado undefined.*/

//Resumen:

.find() //es útil cuando necesitas encontrar el primer elemento en un array que cumple con una condición específica.

/*A diferencia de .filter(), que devuelve todos los elementos que coinciden, .find() devuelve solo el 
primer elemento que cumpla con la condición.*/

/*Si no se encuentra ningún elemento, retorna */undefined.

/*Eficiencia:*/ .find()/* es más eficiente y directo que usar*/ .forEach() /*para obtener el primer 
elemento que coincida con una condición.*/

Método .every()

El método .every() en JavaScript evalúa si todos los elementos de un array cumplen con una condición específica. Si todos los elementos cumplen la condición, el método retorna true; de lo contrario, retorna false. Este método es particularmente útil cuando necesitas asegurarte de que cada elemento de un array cumpla con un criterio determinado. 

//Ejemplo detallado con .every():

        const carrito = [
        { nombre: "Monitor 27 Pulgadas", precio: 500 },
        { nombre: "Televisión", precio: 100 },
        { nombre: "Tablet", precio: 200 },
        { nombre: "Audifonos", precio: 300 },
        { nombre: "Teclado", precio: 400 },
        { nombre: "Celular", precio: 700 },
        ];
                
        const resultado = carrito.every((producto) => producto.precio < 500);
                
        console.log(resultado);// Resultado: false (porque "Monitor 27 Pulgadas" y "Celular" tienen un precio mayor o igual a 500)

/*Explicación:

Condición evaluada: En este caso, la condición es*/ producto.precio < 500 /*. Esto significa que todos los productos del carrito 
tener un precio menor a 500 para que el método devuelva true.*/
        
/*Resultado: El método retorna false porque no todos los productos cumplen con la condición; específicamente, el "Monitor 27 Pulgadas" 
el "Celular" tienen precios mayores o iguales a 500.*/

//Usa .every() cuando quieras asegurarte de que todos los elementos de un array cumplan con una condición.

Método .concat()

Unir arreglos en JavaScript es una tarea común que se puede realizar utilizando diferentes métodos como .concat() o el operador spread (...). Ambos permiten combinar múltiples arrays en uno solo, pero ofrecen diferencias en flexibilidad y sintaxis. 

/*El método .concat() se utiliza para combinar dos o más arrays. También puedes añadir elementos individuales además de los arrays.*/

        const meses = ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio", "Julio"];
        const meses2 = ["Agosto", "Septiembre"];
        const meses3 = ["Octubre", "Noviembre", "Diciembre"];
                  
        const resultado = meses.concat(meses2, meses3, "otro mes");
                  
        console.log(resultado); /*Resultado: ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio",
        "Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre", "otro mes"]*/

/*Explicación:

/*Combina arrays: */ meses.concat(meses2, meses3) /*une los tres arrays en el orden en que se pasan como argumentos.*/

//Añade elementos: También puedes incluir valores adicionales, como "otro mes".

/*El operador spread (...) es una alternativa más moderna y flexible para combinar arrays.
Permite "expandir" el contenido de un array dentro de otro, lo que lo hace más legible y dinámico.*/

        const resultado2 = [...meses, ...meses2, ...meses3, "otros meses"];

        console.log(resultado2);/*Resultado: ["Enero", "Febrero", "Marzo", "Abril", "Mayo", "Junio",
        "Julio", "Agosto", "Septiembre", "Octubre", "Noviembre", "Diciembre", "otros meses"]*/

/*Conclusión:

Usa .concat() si prefieres una opción más tradicional o estás trabajando en entornos que no soportan ES6.
Opta por spread operator (...) para un código más limpio y moderno, especialmente si ya utilizas otras características de ES6+.*/

Método .reverse()

El método .reverse() en JavaScript invierte el orden de los elementos de un array, modificando el array original. El primer elemento pasa a ser el último y viceversa. 

//Ejemplo:

        const numeros = [1, 2, 3, 4, 5];

        console.log("Array original:", numeros); //Array original: [1, 2, 3, 4, 5]
                  
        numeros.reverse();
                  
        console.log("Array invertido:", numeros); // Array invertido: [5, 4, 3, 2, 1]

/*Si necesitas conservar el array original y también tener una versión invertida, puedes combinar
.reverse() con el operador spread (...) para crear una copia antes de invertir:*/

        const copiaInvertida = [...numeros].reverse();
        console.log(copiaInvertida); // [5, 4, 3, 2, 1]
        console.log(numeros); // [1, 2, 3, 4, 5]
