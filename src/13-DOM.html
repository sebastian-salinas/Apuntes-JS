DOM

Acceso a Elementos del DOM

El DOM (Document Object Model) es una representación estructurada de un documento HTML que permite a los lenguajes de programación, acceder y manipular el contenido, estructura y estilo de una página web. Es como un puente entre el código HTML y el navegador, donde cada elemento del documento (etiquetas, atributos, texto) se convierte en un objeto que puede ser modificado dinámicamente. 

/*En este código, se muestran formas de acceder a distintas partes del DOM (Document Object Model)
mediante propiedades del objeto global document:*/

1. /*document: 
      Devuelve todo el contenido del documento HTML, es decir, el DOM completo. Esto incluye las etiquetas,
      elementos, y nodos del documento cargado.*/

        elemento = document;
        console.log(elemento); // Muestra el DOM completo en la consola

2. /*document.head: 
      Devuelve el contenido del head del documento HTML. El head suele contener metadatos como el título 
      del documento, enlaces a estilos CSS y scripts.*/

        elemento = document.head;
        console.log(elemento); // Muestra el contenido del head
      
3. /*document.body: 
      Devuelve el contenido del body del documento HTML. Aquí se encuentra la mayor parte del 
      contenido visible de la página web.*/

        elemento = document.body;
        console.log(elemento); // Muestra el contenido del body

4. /*document.classList: 
      Accede a las clases asociadas a un elemento del DOM, generalmente usadas para manipular
      estilos y comportamientos dinámicos.

      Nota: document.classList no es válido directamente en document ya que document no tiene clases, pero 
      sí lo es para elementos individuales. Podría usarse en un elemento específico como:
      document.querySelector("div").classList.*/

        elemento = document.querySelector("div").classList;
        console.log(elemento); // Lista de clases del primer div

/*Conclusión

Estas propiedades son fundamentales para interactuar y manipular el DOM de una página web.
Permiten acceder a elementos clave del documento HTML para modificar su estructura, contenido, o estilos dinámicamente.*/

Uso de getElementsByClassName()

El método getElementsByClassName() permite seleccionar elementos del DOM que tienen una clase específica. Este método devuelve una colección en vivo de todos los elementos que coinciden con la clase proporcionada. 

//Ejemplo:
        const header = document.getElementsByClassName("header");
        console.log(header);
        
/*Se selecciona y muestra en la consola todos los elementos con la clase header. Si existe solo un 
elemento con esta clase, estará como único en la colección.*/

//Seleccionar múltiples elementos:

        const hero = document.getElementsByClassName("hero");
        console.log(hero);

/*Explicación: Si hay varios elementos con la clase hero, todos se incluirán en una colección. 
Puedes acceder a ellos por su índice como si fuera un array.*/

//Puntos importantes:

1. /*El resultado no es un array, sino una colección HTML. Aunque se parece a un array, no tiene 
métodos como .map o .filter.*/

2. /*Es una colección en vivo, lo que significa que si se agrega o elimina un elemento con la clase
seleccionada después de ejecutar el código, la colección se actualiza automáticamente.*/

3. /*Este método solo funciona para clases. Si necesitas seleccionar por ID, usa getElementById,
y si necesitas más flexibilidad, usa querySelector o querySelectorAll.*/

Uso de getElementsById()

El método getElementsById() permite seleccionar un elemento único del DOM utilizando su atributo id. Es una de las formas más comunes y rápidas para acceder a elementos específicos, ya que el atributo id debe ser único en el documento. 

//Ejemplo:
        const formulario = document.getElementById("formulario");
        console.log(formulario);

/*Aquí seleccionamos el elemento que tiene el atributo id="formulario".

El método devuelve el elemento completo, incluyendo todos sus atributos, contenido y nodos hijos.

Si el elemento existe, se muestra en la consola. Si no, devuelve null.*/

//Características principales de getElementById:

1. //Acceso directo y rápido: Este método es uno de los más eficientes para acceder a elementos en el DOM.

2. //Unicidad del ID: Dado que los id deben ser únicos en el documento, siempre devuelve un único elemento (o null si no lo encuentra).

3. //Compatibilidad: Funciona en todos los navegadores modernos y es ampliamente utilizado.

/*Consejo práctico:

Si necesitas seleccionar elementos con más flexibilidad (por clase, atributo, o combinaciones),
considera usar querySelector o querySelectorAll. Sin embargo, para elementos con ID únicos, 
getElementById sigue siendo una excelente opción.*/

Uso de querySelector()

El método querySelector() selecciona el primer elemento del DOM que coincide con un selector CSS especificado. Es una herramienta versátil y poderosa para trabajar con el DOM, ya que permite aplicar los mismos selectores que en CSS. 

//Ejemplo:
        const card = document.querySelector(".card");
        console.log(card);
        
// Cambiar el estilo del primer elemento con la clase "card"
        card.style.backgroundColor = "red";
                  
//Selectores específicos como en CSS:

        const info = document.querySelector(".premium .info");  
        console.log(info);

/*Este código selecciona el primer elemento con la clase info que está dentro de un contenedor con la clase premium.*/

//Seleccionar elementos específicos:

        const segundoCard = document.querySelector("section.hospedaje .card:nth-child(2)");
        console.log(segundoCard);

/*Se selecciona el segundo hijo (nth-child(2)) con la clase card, dentro de un section con la clase hospedaje.*/

//Ventajas de querySelector:

1. //Flexibilidad: Permite usar cualquier selector CSS (clases, ID, pseudo-clases, combinadores, etc.).

2. //Versatilidad: Funciona para cualquier tipo de elemento y selector válido en CSS.

3. //Conveniencia: Ideal para seleccionar elementos específicos o realizar selecciones complejas.

Uso de querySelectorAll()

El método querySelectorAll() permite seleccionar todos los elementos que coincidan con un selector CSS y devuelve una lista de nodos. Esto es útil para aplicar cambios a varios elementos a la vez. 

//Ejemplo:
// Agrega un evento de clic a todo el documento
        document.addEventListener("click", cambiarColor);

        function cambiarColor() {
// Selecciona todos los elementos con la clase "card"
        const cardsTotales = document.querySelectorAll(".card");

// Recorre cada tarjeta y cambia su color de fondo
        cardsTotales.forEach((card) => {
          if (card.style.backgroundColor === "blue") {
            card.style.backgroundColor = "white"; // Vuelve al color original
          } else {
            card.style.backgroundColor = "blue"; // Cambia a azul
          }
        });
      }

Uso de innerHTML

El método innerHTML permite acceder o modificar el contenido HTML de un elemento en el DOM. Es útil cuando queremos cambiar dinámicamente textos o agregar elementos dentro de un contenedor sin recargar la página. 

//Ejemplo:
        const mensajeDiv = document.querySelector("#mensaje");

// Cambiar el contenido del div con HTML dinámico
        mensajeDiv.innerHTML = "
¡Bienvenido!
Gracias por visitar nuestro sitio.
";
                  
//Conclusión:

1.// innerHTML es util para insertar contenido con formato
2.//Si el contenido proviene de un usuario, debería ser validado para evitar ataques de inyección de código.
3.//Si solo se necesita cambiar texto sin etiquetas HTML, es mejor usar .textContent para evitar interpretaciones inesperadas.

Uso de classList.add()

El método classList.add() permite agregar una o varias clases CSS a un elemento. Es una forma sencilla de modificar la apariencia de un elemento de manera dinámica sin tener que manipular directamente la propiedad class. 

//Ejemplo: Añadir una clase a un botón al hacer clic

        const boton = document.querySelector("button");

        boton.addEventListener("click", () => {
          boton.classList.add("activo");
        });

//Explicación:

1.//document.querySelector("button") → Selecciona el primer botón en el documento.
2.//boton.addEventListener("click", () => {...}) → Escucha el evento de clic en el botón.
3./*boton.classList.add("activo") → Al hacer clic, agrega la clase "activo" al botón,
    lo que puede cambiar su estilo según esté definido en CSS.*/

//Ejemplo con múltiples clases:

        boton.classList.add("activo", "destacado");

//Esto agregará tanto la clase "activo" como la clase "destacado" al elemento.

Traversing de DOM en JavaScript

El traversing de DOM (recorrido del DOM) hace referencia a la forma en que puedes acceder a los elementos y nodos dentro del árbol del DOM. En JavaScript, todos los elementos HTML están conectados al objeto global document, y a través de diferentes métodos y propiedades, puedes navegar y manipular estos elementos. 

//Ejemplo: Accediendo y modificando elementos en el DOM

// Seleccionar la primera tarjeta con la clase 'card'
        const card3 = document.querySelector(".card");
                  
// Cambiar el texto de un hijo específico dentro de la tarjeta
        console.log(
          (card3.children[1].children[1].textContent = "Nuevo heading desde traversing")
        );

//Explicación:

1.//document.querySelector(".card") → Selecciona el primer elemento con la clase card.
2.//card3.children[1] → Accede al segundo hijo de la tarjeta (porque el índice comienza en 0).
3.//card3.children[1].children[1] → Dentro de ese segundo hijo, selecciona el segundo elemento.
4.//.textContent → Cambia el texto de este segundo elemento a "Nuevo heading desde traversing".

// Cambiar la fuente de la imagen dentro de la tarjeta
        card3.children[0].src = "img/hacer4.jpg";

//Explicación:

1.//card3.children[0] → Accede al primer hijo de la tarjeta (probablemente una imagen).
2.//.src → Modifica la fuente de la imagen, cambiándola a "img/hacer4.jpg".

/*Puntos clave del traversing de DOM:

  children: Accede a los hijos directos de un elemento, devolviendo una colección de nodos (como un arreglo).
  textContent: Cambia o obtiene el texto dentro de un nodo.
  src: Modifica la fuente de una imagen o un recurso dentro del DOM.

¿Por qué es útil el traversing de DOM?

  El traversing te permite navegar por la estructura jerárquica de un documento HTML, 
  accediendo y manipulando los elementos según su posición o relación con otros elementos en el árbol del DOM.*/

Eliminar elementos del DOM en JavaScript

En JavaScript, puedes eliminar elementos del DOM utilizando dos métodos principales. Esto te permite quitar elementos de la página web de manera dinámica y modificar la estructura del documento según sea necesario. 

//Ejemplo 1: Eliminar un elemento completo con remove()

        const card6 = document.querySelector(".card");
        card6.remove(); // Elimina el nodo completo del DOM

//Explicación:

1.//document.querySelector(".card"): Selecciona el primer elemento con la clase card.
2./*.remove(): Elimina completamente el elemento card6 del DOM, quitando tanto el nodo
    como todos sus elementos secundarios (si los tiene).*/

//Ejemplo 2: Eliminar un hijo específico usando removeChild()

        const navegacion = document.querySelector(".navegacion");
        navegacion.removeChild(navegacion.children[1]); // Elimina el segundo hijo de 'navegacion'

//Explicación:

1.//document.querySelector(".navegacion"): Selecciona el primer elemento con la clase navegacion.
2.//navegacion.children[1]: Accede al segundo hijo de este elemento, ya que el índice comienza desde 0.
3./*navegacion.removeChild(...): Elimina el hijo seleccionado de su elemento padre 
    (en este caso, el segundo hijo de la barra de navegación).*/

Generar HTML dinámicamente desde JavaScript

En JavaScript, es posible crear y modificar elementos HTML de manera dinámica utilizando el DOM. Esto te permite agregar, modificar o eliminar contenido en la página sin tener que recargarla. A continuación, te explico cómo generar y manipular HTML directamente desde tu código JavaScript. 

//Ejemplo: Crear y agregar elementos HTML

// Crear un nuevo párrafo
        const parrafo1 = document.createElement("p");
        parrafo1.textContent = "Cena";  // Asignar contenido de texto al párrafo
        parrafo1.classList.add("categoria", "concierto");  // Añadir clases CSS
                  
        const parrafo2 = document.createElement("p");
        parrafo2.textContent = "Restaurant Italiano";
        parrafo2.classList.add("titulo");
                  
        const parrafo3 = document.createElement("p");
        parrafo3.textContent = "30€ por persona";
        parrafo3.classList.add("precio");
                  
// Crear un contenedor div para la información
        const info = document.createElement("div");
        info.classList.add("info");
        info.appendChild(parrafo1);  // Añadir párrafo al contenedor
        info.appendChild(parrafo2);
        info.appendChild(parrafo3);
                  
// Crear una imagen
        const imagen = document.createElement("img");
        imagen.src = "img/hacer3.jpg";  // Asignar la fuente de la imagen
                  
// Crear un card para contener la imagen y la info
        const card9 = document.createElement("div");
        card9.classList.add("card");  // Añadir la clase 'card' al div
                  
// Asignar la imagen y el contenedor de información al card
        card9.appendChild(imagen);
        card9.appendChild(info);
                  
// Insertar el card en el HTML
        const contenedor = document.querySelector(".hacer .contenedor-cards");
        contenedor.insertBefore(card9, contenedor.children[0]);  // Insertar el card en la primera posición

//Explicación paso a paso:

//Crear elementos:
        
  1.//document.createElement(): Este método se utiliza para crear un nuevo elemento del tipo especificado, como p, div, img, etc.
  2.//Ejemplo: const parrafo1 = document.createElement("p"); crea un nuevo párrafo p.

//Asignar contenido:
        
  1.//textContent: Se utiliza para agregar texto al nuevo elemento.
  2.//Ejemplo: parrafo1.textContent = "Cena"; asigna el texto "Cena" al párrafo.

//Agregar clases:
        
  1.//classList.add(): Permite añadir una o más clases CSS al elemento creado, lo que es útil para aplicar estilos.
  2.//Ejemplo: parrafo1.classList.add("categoria", "concierto"); añade las clases categoria y concierto al párrafo.

//Anidar elementos:
        
  1.//appendChild(): Agrega un elemento como hijo del elemento seleccionado.
  2.//Ejemplo: info.appendChild(parrafo1); inserta el párrafo dentro del div de clase info.

//Insertar en el HTML:
        
  1.//insertBefore(): Inserta el nuevo elemento en una posición específica dentro de su contenedor, antes del primer hijo.
  2./*Ejemplo: contenedor.insertBefore(card9, contenedor.children[0]); 
    inserta el nuevo "card" antes del primer elemento dentro del contenedor.*/

//Ventajas de crear HTML dinámicamente:

1.//Dinamismo: Puedes crear contenido dinámicamente sin necesidad de escribir el HTML manualmente.
2.//Interactividad: Puedes añadir contenido basado en las acciones del usuario o en datos provenientes de una API.
3./*Actualización en tiempo real: Es ideal para actualizar o agregar elementos en una página 
    sin recargarla (como en las aplicaciones web interactivas).*/

Ejemplo avanzado de JavaScript

En este ejemplo veremos cómo seleccionar elementos del DOM, agregar eventos, y manipular las clases de los elementos para mostrar u ocultar contenido de manera dinámica. El código proporciona una funcionalidad donde al hacer clic en un botón, se muestra u oculta un pie de página (footer) y cambia el texto del botón. 

// Seleccionar el botón y el pie de página
        const btnFooter = document.querySelector(".btn-flotante");
        const footer = document.querySelector(".footer");
                  
// Añadir evento de clic al botón
        btnFooter.addEventListener("click", mostrarOcultarFooter);
                  
        function mostrarOcultarFooter() {
// Si el footer tiene la clase 'activo', la quitamos
          if (footer.classList.contains("activo")) {
            this.classList.remove("activo"); // Quitar 'activo' del botón
            footer.classList.remove("activo"); // Quitar 'activo' del pie de página
            this.textContent = "Idioma y Moneda"; // Cambiar el texto del botón
          } else {
                      // Si no tiene la clase 'activo', la agregamos
            this.classList.add("activo"); // Añadir 'activo' al botón
            footer.classList.add("activo"); // Añadir 'activo' al pie de página
            this.textContent = "X Cerrar"; // Cambiar el texto del botón
          }
// Alternativa con toggle (comentada)
// footer.classList.toggle("activo");
// btnFooter.classList.toggle("activo");
        }

//Explicación paso a paso:

//Seleccionar elementos del DOM:
        
  1.//Usamos document.querySelector() para seleccionar el botón flotante (btn-flotante) y el pie de página (footer).
  2.//Ejemplo: const btnFooter = document.querySelector(".btn-flotante"); selecciona el botón con la clase btn-flotante.

//Agregar un evento de click:
        
  1./*Se utiliza addEventListener() para escuchar el evento click en el botón flotante. 
  Cuando el usuario hace clic en el botón, se ejecuta la función mostrarOcultarFooter()./*
  2./*Ejemplo: btnFooter.addEventListener("click", mostrarOcultarFooter); asigna la 
  función mostrarOcultarFooter al evento de clic./*

//Manipulación de clases CSS:
        
  1./*Dentro de la función mostrarOcultarFooter(), se verifica si el footer ya tiene la clase activo
  mediante classList.contains().*/
  2./*Si tiene la clase, se elimina la clase activo tanto del footer como del botón utilizando 
  classList.remove(), y se cambia el texto del botón a "Idioma y Moneda".*/
  3./*Si no tiene la clase, se añade la clase activo tanto al footer como al botón con 
  classList.add(), y se cambia el texto del botón a "X Cerrar".*/

//lternativa usando classList.toggle():
        
  1./*La función está comentada, pero classList.toggle() es una forma más sencilla de agregar o quitar
  una clase dependiendo de si ya está presente o no. Al usar toggle, no es necesario comprobar si la 
  clase existe o no previamente. Solo se pasa la clase y el método se encarga de añadirla o eliminarla según corresponda.*/
  2.//Ejemplo: footer.classList.toggle("activo"); cambiaría el estado de la clase activo de manera automática.

//Resumen:
        
  1.//querySelector(): Se utiliza para seleccionar elementos del DOM.
  2.//addEventListener(): Permite escuchar y manejar eventos como click.
  3.//classList.add() y classList.remove(): Se utilizan para añadir o eliminar clases de los elementos.
  4.//classList.contains(): Verifica si un elemento tiene una clase específica.
  5.//classList.toggle(): Alterna (añade o elimina) una clase en el elemento de manera automática.
