DOM

Acceso a Elementos del DOM

El DOM (Document Object Model) es una representación estructurada de un documento HTML que permite a los lenguajes de programación, acceder y manipular el contenido, estructura y estilo de una página web. Es como un puente entre el código HTML y el navegador, donde cada elemento del documento (etiquetas, atributos, texto) se convierte en un objeto que puede ser modificado dinámicamente. 

/*En este código, se muestran formas de acceder a distintas partes del DOM (Document Object Model)
mediante propiedades del objeto global document:*/

1. /*document: 
      Devuelve todo el contenido del documento HTML, es decir, el DOM completo. Esto incluye las etiquetas,
      elementos, y nodos del documento cargado.*/

        elemento = document;
        console.log(elemento); // Muestra el DOM completo en la consola

2. /*document.head: 
      Devuelve el contenido del head del documento HTML. El head suele contener metadatos como el título 
      del documento, enlaces a estilos CSS y scripts.*/

        elemento = document.head;
        console.log(elemento); // Muestra el contenido del head
      
3. /*document.body: 
      Devuelve el contenido del body del documento HTML. Aquí se encuentra la mayor parte del 
      contenido visible de la página web.*/

        elemento = document.body;
        console.log(elemento); // Muestra el contenido del body

4. /*document.classList: 
      Accede a las clases asociadas a un elemento del DOM, generalmente usadas para manipular
      estilos y comportamientos dinámicos.

      Nota: document.classList no es válido directamente en document ya que document no tiene clases, pero 
      sí lo es para elementos individuales. Podría usarse en un elemento específico como:
      document.querySelector("div").classList.*/

        elemento = document.querySelector("div").classList;
        console.log(elemento); // Lista de clases del primer div

/*Conclusión

Estas propiedades son fundamentales para interactuar y manipular el DOM de una página web.
Permiten acceder a elementos clave del documento HTML para modificar su estructura, contenido, o estilos dinámicamente.*/

Uso de getElementsByClassName()

El método getElementsByClassName() permite seleccionar elementos del DOM que tienen una clase específica. Este método devuelve una colección en vivo de todos los elementos que coinciden con la clase proporcionada. 

//Ejemplo:
        const header = document.getElementsByClassName("header");
        console.log(header);
        
/*Se selecciona y muestra en la consola todos los elementos con la clase header. Si existe solo un 
elemento con esta clase, estará como único en la colección.*/

//Seleccionar múltiples elementos:

        const hero = document.getElementsByClassName("hero");
        console.log(hero);

/*Explicación: Si hay varios elementos con la clase hero, todos se incluirán en una colección. 
Puedes acceder a ellos por su índice como si fuera un array.*/

//Puntos importantes:

1. /*El resultado no es un array, sino una colección HTML. Aunque se parece a un array, no tiene 
métodos como .map o .filter.*/

2. /*Es una colección en vivo, lo que significa que si se agrega o elimina un elemento con la clase
seleccionada después de ejecutar el código, la colección se actualiza automáticamente.*/

3. /*Este método solo funciona para clases. Si necesitas seleccionar por ID, usa getElementById,
y si necesitas más flexibilidad, usa querySelector o querySelectorAll.*/

Uso de getElementsById()

El método getElementsById() permite seleccionar un elemento único del DOM utilizando su atributo id. Es una de las formas más comunes y rápidas para acceder a elementos específicos, ya que el atributo id debe ser único en el documento. 

//Ejemplo:
        const formulario = document.getElementById("formulario");
        console.log(formulario);

/*Aquí seleccionamos el elemento que tiene el atributo id="formulario".

El método devuelve el elemento completo, incluyendo todos sus atributos, contenido y nodos hijos.

Si el elemento existe, se muestra en la consola. Si no, devuelve null.*/

//Características principales de getElementById:

1. //Acceso directo y rápido: Este método es uno de los más eficientes para acceder a elementos en el DOM.

2. //Unicidad del ID: Dado que los id deben ser únicos en el documento, siempre devuelve un único elemento (o null si no lo encuentra).

3. //Compatibilidad: Funciona en todos los navegadores modernos y es ampliamente utilizado.

/*Consejo práctico:

Si necesitas seleccionar elementos con más flexibilidad (por clase, atributo, o combinaciones),
considera usar querySelector o querySelectorAll. Sin embargo, para elementos con ID únicos, 
getElementById sigue siendo una excelente opción.*/

Uso de querySelector()

El método querySelector() selecciona el primer elemento del DOM que coincide con un selector CSS especificado. Es una herramienta versátil y poderosa para trabajar con el DOM, ya que permite aplicar los mismos selectores que en CSS. 

//Ejemplo:
        const card = document.querySelector(".card");
        console.log(card);
        
// Cambiar el estilo del primer elemento con la clase "card"
        card.style.backgroundColor = "red";
                  
//Selectores específicos como en CSS:

const info = document.querySelector(".premium .info");  
console.log(info);

/*Este código selecciona el primer elemento con la clase info que está dentro de un contenedor con la clase premium.*/

//Seleccionar elementos específicos:

const segundoCard = document.querySelector("section.hospedaje .card:nth-child(2)");
console.log(segundoCard);

/*Se selecciona el segundo hijo (nth-child(2)) con la clase card, dentro de un section con la clase hospedaje.*/

//Ventajas de querySelector:

1. //Flexibilidad: Permite usar cualquier selector CSS (clases, ID, pseudo-clases, combinadores, etc.).

2. //Versatilidad: Funciona para cualquier tipo de elemento y selector válido en CSS.

3. //Conveniencia: Ideal para seleccionar elementos específicos o realizar selecciones complejas.

Reemplazar y Extraer Parte de una Cadena

Usamos replace() para reemplazar texto y slice() o substring() para extraer porciones de una cadena. 


        let puntaje = 5;

        puntaje++; // ejecuta el comando y luego lo incrementa
                  
        ++puntaje; // incrementa el puntaje y luego ejecuta el comando
                  
        puntaje += 2; // suma 2 a puntaje y luego ejecuta el comando
                  
        console.log(puntaje);
