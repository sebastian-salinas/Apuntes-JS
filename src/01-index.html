Introducción a JavaScript

¿Qué es JavaScript?

JavaScript es un lenguaje de programación versátil que se utiliza principalmente para agregar interactividad a sitios web. Es uno de los pilares del desarrollo web, junto con HTML y CSS.

Ejemplo básico de JavaScript

A continuación, se muestra un ejemplo básico que imprime "¡Hola, mundo!" en la consola del navegador: 


          // Este es un comentario en JavaScript
          console.log('¡Hola, mundo!');

Características principales

  • Es ejecutado por el navegador en el lado del cliente.
  • Soporta programación orientada a objetos, funcional y basada en eventos.
  • Se utiliza tanto para desarrollo front-end como back-end (con Node.js).
