Numbers

Declaración y uso de numbers

JavaScript solo tiene un tipo de número. Los números pueden escribirse con o sin decimales: 

// Creacion de numero

          const numero1 = 30;
                  
          console.log(numero1);

          const numero2 = 30.14;
                  
          console.log(numero2);

Operadores aritméticos basicos

Los operadores aritméticos básicos en JavaScript se usan para realizar cálculos matemáticos simples: 

//Operadores aritméticos basicos

        const numero1 = 30;
        const numero2 = 20;
                  
        let resultado;
                  
//Suma
                  
        resultado = numero1 + numero2;
                  
//resta
                  
        resultado = numero1 - numero2;
                  
//multiplicación

        resultado = numero1 * numero2;
                  
//división

        resultado = numero1 / numero2;
                  
//Modulo

        resultado = numero1 % numero2;
                  
        console.log(resultado);

Orden de ejecución

El orden de ejecución de las operaciones sigue las reglas de precedencia de operadores, donde las operaciones dentro de paréntesis se evalúan primero, seguidas por la multiplicación/división, y finalmente la suma/resta. Si varios operadores tienen la misma precedencia, se evalúan de izquierda a derecha. 

//Orden de ejecución de operadores

        let resultado;
                  
        resultado = 2 + 3 * 4;
                  
        console.log(resultado);
                  
        resultado = 2 + 3 * 4 - 1;
                  
        console.log(resultado);
                  
        resultado = 2 + 3 * 4 - 1 / 2;
                  
        console.log(resultado);
                  
        resultado = 2 + 3 * 4 - (1 / 2) * 3;
                  
        console.log(resultado);
                  
        resultado = 2 + 3 * 4 - (1 / 2) * 3 + 5;
                  
        console.log(resultado);
                  
        resultado = 2 + 3 * 4 - (1 / 2) * 3 + 5 * 6;
                  
        console.log(resultado);

Incrementos y decrementos

Los incrementos en JavaScript permiten aumentar el valor de una variable de forma sencilla, ya sea en pasos de uno (i++ o ++i) o en valores personalizados (i += n). 

//Incremento

        let i = 5;

// Incremento postfijo

        console.log(i++); // Muestra 5, luego i se convierte en 6
        console.log(i);   // Muestra 6
                  
// Incremento prefijo

        console.log(++i); // Incrementa i primero (7), luego lo muestra
                  
// Incremento personalizado

        i += 3;           // Suma 3 al valor actual de i (ahora 10)
        console.log(i);   // Muestra 10

//Decremento

        let i = 10;

// Decremento postfijo

        console.log(i--); // Muestra 10, luego i se convierte en 9
        console.log(i);   // Muestra 9

// Decremento prefijo

        console.log(--i); // Decrementa i primero (8), luego lo muestra

// Decremento personalizado

        i -= 2;           // Resta 2 al valor actual de i (ahora 6)
        console.log(i);   // Muestra 6

Convertir strings a numbers

En JavaScript se pueden utilizar los metodos parseInt() y parseFloat(), para convertir numeros de texto a números. 

// Usando parseInt para convertir texto a entero

        let textoEntero = "42";
        let numeroEntero = parseInt(textoEntero);
        console.log(numeroEntero); // 42
                  
// Usando parseFloat para convertir texto a número decimal

        let textoDecimal = "42.75";
        let numeroDecimal = parseFloat(textoDecimal);
        console.log(numeroDecimal); // 42.75
                  
// Intentando convertir texto no numérico

        let textoInvalido = "abc";
        console.log(parseInt(textoInvalido)); // NaN
        console.log(parseFloat(textoInvalido)); // NaN

Objeto Math

El objeto Math. proporciona propiedades y métodos para realizar operaciones matemáticas comunes, como redondeos, cálculos de potencias, raíces, y más. Es un objeto estático, lo que significa que no necesitas instanciarlo para usar sus métodos. 

//Objeto math, se accede con la palabra reservada Math y luego un punto y el nombre del método

        let resultado;
                  
        resultado = Math.PI;
                  
//Redondear hacia
                  
        resultado = Math.round(2.8);
                  
//Redondear hacia arriba
                  
        resultado = Math.ceil(2.78);
                  
//Redondear hacia abajo
                  
        resultado = Math.floor(2.78);
                  
//Raiz cuadrada
                  
        resultado = Math.sqrt(2.78);
                  
// Valor absoluto (ignora el negativo)
                  
        resultado = Math.abs(-200);
                  
//Potencia
                  
        resultado = Math.pow(2, 3);
                  
//Minimo
                  
        resultado = Math.min(2, 3, 4, 8, 1, 4, 6, 9);
                  
//Máximo
                  
        resultado = Math.max(2, 3, 4, 8, 1, 4, 6, 9);
                  
// Aleatorio
                  
        let random = Math.random() * 100;
                  
        let redondear = Math.round(random);
                  
        console.log(redondeear);
                  
//Redondear entre 1 y 50
                  
        resultado = Math.round(Math.random() * 50);
                  
        console.log(resultado);
