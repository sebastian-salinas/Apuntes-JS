Strings

Declaración y uso de Numbers

Strings o Cadenas de Texto son secuencias de caracteres que se pueden asignar a variables 

//Strings o cadenas de texto
          
          const nombreProducto = "Monitor de 55 pulgadas";
          console.log(nombreProducto);
          
          const nombreProducto2 = String("Monitor de 65 pulgadas");
          console.log(nombreProducto2);
          
          const nombreProducto3 = new String("Monitor de 75 pulgadas");
          console.log(nombreProducto3);
          
//Escapar caracteres o comillas se representa con una barra inversa
          
          const nombreProducto4 = '"Monitor de 75 pulgadas"';
          console.log(nombreProducto4);

Métodos de Cadenas de Texto

Existen varios métodos útiles para trabajar con cadenas en JavaScript, como obtener la longitud de un string, buscar la posición de un caracter, y más. 

//Longitud de mi string
          const longitudProducto = "Monitor de 32 pulgadas";
          
          console.log(longitudProducto.length);
          
//Metodo para conocer la posicion de un caracter en un string
          
          console.log(longitudProducto.indexOf("pulgadas"));
          
//Metodo para conocer si un string incluye un caracter solicitado
          
          console.log(longitudProducto.includes("pulgadas"));
          
          let array = ["Samsung", "Apple", "Huawei", "LG", "Xiaomi"];
          
          if (array.includes("Xiaomi")) {
            console.log("El movil que busca se encuentra en el array");
          } else {
            console.log("No se encuentra el movil en el array");
          }

Concatenación de Cadenas

Puedes concatenar cadenas de texto y variables usando plantillas literales, lo que hace que tu código sea más limpio y fácil de leer. 

//Concatenacion de strings o variables
          
          const product = "Computadora";
          const precio = 1000;
          const marca = "Acer";
          
          console.log(
            \`La \${product} que se encuentra en stock, es de la marca \${marca} y tiene un valor de \${precio}€\`
          );

Métodos para Eliminar Espacios en Blanco

Utilizamos trimStart() y trimEnd() para eliminar espacios en blanco al principio y al final de una cadena. 

// Metodos para eliminar los espacios en blanco
          
          const mes = "                         Noviembre       ";
          
          console.log(mes.length);
          
//Eliminar espacios en blanco al inicio y al final con trimStart y trimEnd
          
          console.log(mes.trimStart().trimEnd()); //Eliminar espacios en blanco al inicio

Reemplazar y Extraer Parte de una Cadena

Usamos replace() para reemplazar texto y slice() o substring() para extraer porciones de una cadena. 

//Metodo replace, se utiliza para reemplazar el texto de una cadena
          
          const diaDeLaSemana = "Lunes";
          
          console.log(diaDeLaSemana.replace("Lunes", "Martes"));
          
//Metodo slice, se utiliza para extraer una porcion de la cadena de texto
//El primer parametro es el indice de inicio, el segundo parametro es el indice final el cual no se muestra

          console.log(diaDeLaSemana.slice(0, 3));
          
//Metodo substring, similar a slice, pero interpreta si pasamos los parametros en el orden inverso

          console.log(diaDeLaSemana.substring(3, 0));

Otros Métodos de Cadenas

Aquí se encuentran otros métodos útiles como charAt(), repeat(), split(), entre otros. 

//Metodo charAt, se utiliza para obtener un caracter en especifico de la cadena de texto

          console.log(diaDeLaSemana.charAt(0));
          
//Metodo repeat, se utiliza para repetir una cadena de texto

          const oferta = " Esta en promoción".repeat(3);
          //Si le pasamos como parametro un numero decimal, se convertirá a entero
          const oferta2 = " Esta en promoción".repeat(3.45);
          
          console.log(oferta);
          console.log(oferta2);
          
//Metodo split, se utiliza para dividir una cadena de texto en varias partes
          
          const estudio = "Estoy aprendiendo JavaScript";
          
          console.log(estudio.split(" "));
          
          const twitter = "Aprendiendo JavaScript #JSModernoConJuan";
          
          console.log(twitter.split("#"));

Métodos de Caso y Conversión de Tipos

Podemos convertir cadenas a mayúsculas o minúsculas usando toUpperCase() y toLowerCase(), o convertir números a texto con toString(). 

//Metodos toUpperCase y toLowerCase, se utilizan para convertir una cadena de texto a mayúsculas o minúsculas
          
          const producto = "Monitor de 55 pulgadas";
          
          console.log(producto.toUpperCase());
          console.log(producto.toLowerCase());
          
          const email = "CORREO@DOMINIO.COM";
          
          console.log(email.toLowerCase());
          
//Convertir un número a texto
          
          const valor = 1200;
          
          console.log(valor.toString());
          
          console.log(typeof valor.toString());
