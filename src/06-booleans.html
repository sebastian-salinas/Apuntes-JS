Booleans

Declaración y uso de Booleans

Los booleanos son valores lógicos que pueden ser verdaderos o falsos, los booleanos se utilizan para evaluar condiciones y tomar decisiones. 

// Declaración y uso de Booleans

        const boolean1 = true;
        const boolean2 = false;

        console.log(boolean1); // true
        console.log(boolean2); // false

        console.log(boolean1 == boolean2); // false
        console.log(boolean1 !== boolean2); // true

//Ejemplo de uso de booleanos

        const autenticado = true;

        if (autenticado) {
          console.log("Autenticado");
        } else {
          console.log("No autenticado");
        }

Buenas practicas utilizando booleanos

Al trabajar con valores booleanos en JavaScript, seguir buenas prácticas asegura que el código sea más claro, eficiente y fácil de mantener. Aquí algunas recomendaciones clave: 

//Evita comparaciones implícitas que puedan generar resultados inesperados. Realiza comparaciones explícitas.

      //Evitar:
        if (i) { // Compara implícitamente si algunaVar es "truthy" o "falsy"
        // Hacer algo
        }

      //Buena práctica:
        if (i === true) { // Comparación explícita con true
        // Hacer algo
        }

//Para evitar errores por la conversión implícita de tipos, siempre usa el operador de igualdad estricta (===).

      //Ejemplo:
        const i = true;
        if (i === true) {
            // Hacer algo
        }
                  
//JavaScript puede interpretar 0 como false y cualquier valor distinto de 0 como true, pero esto puede resultar confuso.

      //Evitar:
        let i = 1; // 1 se interpreta como true
        if (i) {
            // Hacer algo
        }

      //Buena práctica:
        let i = true; // Usa un valor booleano explícito
        if (i) {
            // Hacer algo
        }        

//Utiliza nombres de variables que dejen claro que se trata de un valor booleano, facilitando la comprensión del código.

      //Evitar:
        const x = true;

      //Buena práctica:
        const usuarioLogueado = true;

/*Aunque a veces se utiliza doble negación (!!) para convertir valores en booleanos,
esto puede dificultar la lectura del código.*/

      //Evitar:
        const esValido = !!algunaVariable; // Conversión implícita a booleano

      //Buena práctica:
        const esValido = Boolean(algunaVariable); // Método más claro y explícito

/*Aunque en JavaScript cualquier valor puede ser evaluado como verdadero o falso,
es preferible usar los valores literales true y false para mejorar la claridad.*/

      //Evitar:
        if (i) { // Esto puede ser confuso si algunaVar no es un valor booleano
          // Hacer algo
        }

      //Buena práctica:
        if (i === true) { // Comparación explícita
          // Hacer algo
        }

//Asegúrate de que las condiciones booleanas sean fáciles de entender y que expresen claramente lo que se está evaluando.

      //Ejemplo:
        if (usuarioAutenticado) {
        // Hacer algo
        }
