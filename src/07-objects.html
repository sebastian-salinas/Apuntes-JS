Spread operator y Rest operator

El spread operator (...) en JavaScript se utiliza para expandir o desestructurar elementos de un array o propiedades de un objeto, mientras que el rest operator se emplea para recolectar m煤ltiples elementos o propiedades en un solo array u objeto, permitiendo una sintaxis m谩s concisa y flexible. 

// Spread operator

//Ejemplo con arrays:

        const numeros = [1, 2, 3];
        const nuevosNumeros = [...numeros, 4, 5];
                  
        console.log(nuevosNumeros);  // [1, 2, 3, 4, 5]

//Ejemplo con objetos:

        const persona = { nombre: "Juan", edad: 30 };
        const nuevaPersona = { ...persona, ciudad: "Madrid" };

        console.log(nuevaPersona);  // { nombre: "Juan", edad: 30, ciudad: "Madrid" }

//Rest operator

//Ejemplo con arrays:

        const numeros = [1, 2, 3, 4, 5];
        const [primero, ...resto] = numeros;

        console.log(primero);  // 1
        console.log(resto);    // [2, 3, 4, 5]

//Ejemplo con objetos:

        const persona = { nombre: "Juan", edad: 30, ciudad: "Madrid" };
        const { nombre, ...resto } = persona;

        console.log(nombre);   // "Juan"
        console.log(resto);    // { edad: 30, ciudad: "Madrid" }
